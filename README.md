# DLSF
DLS Fucker  

> 悩めど悩めど答えは得られず、  
> ならば咲かせよ刹那に飾る花。

![](/pics/1.webp)

## Install
```
git clone https://github.com/Ltfjx/DLSF.git
cd DLSF
npm install
```

## Start
```
npm run start
```

## Use DLSF in command-line
[https://github.com/Ltfjx/DLSF-cli](https://github.com/Ltfjx/DLSF-cli)

## Termux Support
```
pkg install nodejs
pkg install git
git clone https://github.com/Ltfjx/DLSF.git
cd DLSF
npm install
npm run start
```

## PWA Support
You can install DLSF as a PWA on your Android device.
